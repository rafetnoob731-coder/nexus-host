Python Quickstart

Deploy your Python bot in minutes. NEXUS HOSTING supports Discord.py, python-telegram-bot, FastAPI, and any Python framework.

1. Create your bot

# main.py
import discord
from discord.ext import commands

bot = commands.Bot(command_prefix='!')

@bot.event
async def on_ready():
    print(f'Logged in as {bot.user}')

@bot.command()
async def ping(ctx):
    await ctx.send('Pong!')

bot.run('YOUR_TOKEN')

2. Add requirements.txt

discord.py==2.4.0
python-dotenv==1.0.1

3. Deploy to NEXUS

Create a project with runtime set to Python, upload your ZIP containing main.py and requirements.txt, and deploy. NEXUS will:

  • ✓ Create a virtual environment
  • ✓ Install all pip dependencies
  • ✓ Set up your entry point
  • ✓ Deploy to global edge nodes

Expected ZIP Structure

my-bot.zip
├── main.py              # Entry point
├── requirements.txt     # Dependencies
├── .env.example         # Env template
└── src/
    └── handlers.py

Laravel Quickstart

Deploy your Laravel bot with Horizon, Queue, Scheduler, and Octane support out of the box.

# composer.json
{
    "require": {
        "laravel/framework": "^11.0",
        "laravel/horizon": "^5.0"
    }
}

Expected ZIP Structure

my-bot.zip
├── artisan
├── composer.json
├── routes/
│   └── bot.php
└── app/
    └── Bots/

JavaScript Quickstart

Deploy Discord.js, node-telegram-bot-api, or any Express-based bot.

// index.js
const { Client, GatewayIntentBits } = require('discord.js');
const client = new Client({ intents: [GatewayIntentBits.Guilds] });

client.on('ready', () => console.log('Ready!'));
client.on('interactionCreate', async interaction => {
    if (interaction.isChatInputCommand() && interaction.commandName === 'ping') {
        await interaction.reply('Pong!');
    }
});

client.login(process.env.TOKEN);

Expected ZIP Structure

my-bot.zip
├── index.js             # Entry point
├── package.json         # Dependencies
├── .env.example
└── src/
    └── commands/

ZIP Deploy Documentation

Upload any bot as a ZIP or tar.gz archive. NEXUS auto-detects the runtime and configures everything.

Auto-Detection Logic

  1. Check for composer.json → Laravel
  2. Check for requirements.txt or setup.py → Python
  3. Check for package.json → JavaScript
  4. If none detected → prompts manual selection

Limits

  • Max file size: 100MB (500MB for Enterprise)
  • Max build time: 15 minutes (60min for Enterprise)
  • Supported formats: .zip, .tar.gz
